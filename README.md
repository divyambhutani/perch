# Perch

## What is Perch?

Perch is a macOS companion app for Claude Code designed to run in the background, providing subtle notifications and status updates without interrupting your workflow.

## Key Features

- Menubar notifications in the notch
- Real-time session status tracking
- Customizable familiars (avatars) for different personas
- Multiple theme options
- Auto-updates via Sparkle

## How to Use

1. Download from GitHub Releases
2. Launch Perch - it will automatically pair with your Claude Code sessions
3. Customize your familiar and theme in Settings
